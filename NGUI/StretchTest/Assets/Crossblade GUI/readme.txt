Crossblade is fantasy traditional interface with more than 40 Elements and 15 Icons. Its style allow you to use it in various game interfaces. Crossblade is good for RPG and MMORPG games for mobile devices or desktops.


The product features:
- Fully sliced layers in .png format
- The complete Photoshop .psd file
- All fonts included
- 46 Elements
- 15 Icons
